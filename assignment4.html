Challenge 5: Accessibility Concerns

The chessboard created in Challenge 2 would be difficult to access. Creating the board as an HTML table with unicode text may cause difficulty when accessing it using certain web browsers or with a screenreader. A better way to create this would have been with an image. This would ensure that it displays correctly in all applicaitons, and an alt text would make it more accessible with a screen reader.
