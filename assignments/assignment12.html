Quote Generator

Press the button to generate a quote.






I extended the machine adding a function to copy the quote to the clipboard. The function is nested inside the API call function. When a quote is generated, a button and text will appear below the quote. Pressing the button copies the quote to the clipboard in the format ["quote" -"author"]. Pressing the button adds text underneath indicating it's been pressed.
